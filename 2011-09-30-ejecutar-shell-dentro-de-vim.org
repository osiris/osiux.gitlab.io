Ayer alguien preguntó en =gcoop= [fn:gcoop] cómo ejecutar un /shell/
como una ventana más de =vim=, tras una rápida búsqueda por la /web/
encontramos un excelente /plugin/ llamado =ConqueTerm= [fn:ConqueTerm]
muy fácil de instalar:

#+BEGIN_SRC sh :results none :exports code

cd ~/.vim
wget -O conque_2.3.vmb https://www.vim.org/scripts/download_script.php?src_id=16279
vim conque_2.3.vmb
:so %
:q

#+END_SRC

Para abrir una ventana con una terminal basta con ejecutar:

#+BEGIN_EXAMPLE

:ConqueTermSplit bash

#+END_EXAMPLE

También se puede invocar a otras aplicaciones:

#+BEGIN_EXAMPLE

:ConqueTermSplit top

#+END_EXAMPLE

Terminé agregando un alias a mi =.vimrc=:

#+BEGIN_EXAMPLE

nmap ,c :ConqueTermSplit bash<CR>

#+END_EXAMPLE

De esta manera cada vez que necesito usar =bash= y no quiero salir de
=vim=, basta con pulsar =,c=. En breve publico todos mis archivos de
configuración. =:-)=
